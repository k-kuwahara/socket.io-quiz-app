# Socket.io_quiz_app
It's an app made by [Socket.IO](http://socket.io/).


## Development Environment
- Node.js
- Socket.IO
- Sass(Scss)
- Bootstrap
- Eslint
- Gulp
- Jade


## How to use
Please this command on your terminal.

```cmd
$ cd socket.io_quiz_app
$ npm install
$ cd src/
$ node server
```
