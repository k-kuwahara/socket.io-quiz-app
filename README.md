# Socket.io_quiz_app
It's an app made by [Socket.IO](http://socket.io/).


## Development Environment
- Node.js
- Socket.IO
- gulp
- gulp-uglify
- gulp-rename
- gulp-sass
- gulp-plumber
- gulp-jshint
- jshint-stylish


## How to use
Please this command on your terminal.

```cmd
$ cd socket.io_quiz_app
$ npm install
$ cd src/
$ node server
```
